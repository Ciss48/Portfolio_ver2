PROJECT • IN PRODUCTION

Electricity Consumption Forecast

XGBoost (Quantile & Tweedie) • Facebook Prophet • Kalman Filter • Python

▸

Forecast commercial electricity consumption at MDD (meter point) level — the most granular level — across 13 provinces, processing several TBs of raw data.

▸

Designed a multi-model architecture with 130 models, segmenting MDDs by industry, consumption size, volatility, and clustering to handle highly heterogeneous customer behavior.

▸

Applied deep feature engineering (lag, diff, pct_change, rolling statistics, group-level features) to capture consumption patterns beyond raw time-series values.

▸

Built an auto dynamic train selection mechanism using Weighted Euclidean distance to identify historically similar periods, avoiding contamination from anomalous data.

▸

Integrated Kalman Smoothing for anomaly detection and correction, and anomaly scoring + ranking to handle forecast outliers without naive averaging.

▸

Applied Facebook Prophet for province-level forecasting as a hierarchical constraint, with regional weight adjustments to account for geographic differences.

▸

Delivered uncertainty quantification via Prophet 80–100% confidence intervals.

The model has been put into production.
